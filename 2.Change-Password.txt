Linux Password Changing
=======================
1) To change the password use the command -
	$ passwd userid

   here, userid is bascially the username can be found using the commmand -
   	$ whoami 


   Example-	
   	$ passwd  your_pass

   it will show - 
   	passwd: Only root can specify a user name.

2) To provide with sudo access -
	
	$ sudo passwd userid

   prompted with change password box having new pass and re-type password
